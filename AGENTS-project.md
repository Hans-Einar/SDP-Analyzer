# AGENTS-project.md

Project-specific agent instructions belong here.

This file is project-owned and is never overwritten by the SDP Toolkit installer.
It is read immediately after the toolkit-managed `AGENTS.md`.

Use it for repository-specific information such as:

- build, test and development commands
- important directories and generated files
- environment constraints
- naming conventions
- deployment or hardware limitations
- project-specific safety rules
- links to authoritative SDP documents

Do not duplicate the complete SDP role and workflow rules here. Do not weaken
slice discipline, independent review, traceability, verification or stop
conditions defined by `AGENTS.md` and the installed SDP skills.
