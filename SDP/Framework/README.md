# SDP Framework

This directory is Toolkit-managed and may be refreshed with
`-ForceManagedFiles`.

Project-specific Mandate, Study, Requirements, Architecture, Design,
Implementation, Sprints, Refactors, Review, Verification and Traceability remain
project-owned.

Core rules:

- repository documents are authoritative
- design horizontally and implement vertically
- one active Slice at a time
- Workers implement bounded slices
- Reviewers use fresh independent context
- completion requires real verification and current traceability
- stop at the Slice boundary
